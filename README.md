Sudulist is a task management program implemented through a single HTML file. 

Key features: Can intelligently identify the time in the sentence and schedule tasks
Quickly set the priority and adjust the time for tasks
Task statistics analysis
Reminder and notification functions
Highly customizable settings
Simple interface with dynamic animations, easy to use
At the same time, there is a cat assistant who can cheer you up while you are working
